# Getting Started with Saman Security

This guide will help you get started with Saman Security.

## Installation

1.  Download the latest release from the [GitHub repository](https://github.com/SamanLabs/Saman-Security/releases).
2.  Upload the plugin to your WordPress site via the "Plugins > Add New" page.
3.  Activate the plugin.

## Configuration

After activating the plugin, you can configure it by navigating to "Security Pilot" in the WordPress admin menu.
