# Developer Guide

This guide provides information for developers who want to extend or customize Saman Security.

## Hooks and Filters

Saman Security provides a number of hooks and filters that you can use to extend its functionality. Please see the [Filter Reference](FILTERS.md) for more information.

## WP-CLI Commands

Saman Security includes a number of WP-CLI commands that you can use to manage the plugin from the command line. Please see the [WP-CLI Command Reference](WP_CLI.md) for more information.
