# WP-CLI Command Reference

This document lists all of the WP-CLI commands available in Saman Security.

## General Commands

-   `wp saman-security status`: Displays the current status of the plugin.
-   `wp saman-security settings get <setting>`: Gets the value of a specific setting.
-   `wp saman-security settings set <setting> <value>`: Sets the value of a specific setting.

## Firewall Commands

-   `wp saman-security firewall list`: Lists all firewall rules.
-   `wp saman-security firewall add <rule>`: Adds a new firewall rule.
-   `wp saman-security firewall delete <rule>`: Deletes a firewall rule.
