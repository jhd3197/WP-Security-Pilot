# Contributing to Saman Security

We love receiving contributions from the community! We want to make it as easy as possible to help us improve Saman Security.

## How to Contribute

There are many ways to contribute, from writing tutorials or blog posts, improving the documentation, submitting bug reports and feature requests or writing code which can be incorporated into Saman Security itself.

### Reporting Bugs

If you find a bug, please open an issue on our [GitHub repository](https://github.com/SamanLabs/Saman-Security/issues). Please include as much detail as possible, including steps to reproduce the bug.

### Suggesting Enhancements

If you have an idea for a new feature or an enhancement to an existing feature, please open an issue on our [GitHub repository](https://github.com/SamanLabs/Saman-Security/issues).

### Pull Requests

We love pull requests! If you'd like to contribute code, please fork the repository and open a pull request. We'll review your pull request as soon as possible.

## Development Setup

1.  Fork the repository and clone it locally.
2.  Install the dependencies: `npm install`
3.  Start the development server: `npm start`
4.  Make your changes.
5.  Submit a pull request.

Thank you for your interest in contributing to Saman Security!
